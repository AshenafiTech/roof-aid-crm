# AI Assistant Instructions

1. After completing any task:
   - Create or update a document inside the `/docs/milestone` folder
   - Use a clear and descriptive markdown file name
   - Document what was done, including:
     - Purpose of the task
     - Steps taken
     - Any important decisions or notes

When writing documentation for any implementation, place it in the appropriate milestone folder.

- Do **not** place documentation directly under the root `/docs` folder.
- Use the corresponding path, such as `docs/milestone-4/`, for features related to Milestone 4.
The documentation should be clean, structured, and easy for other developers to understand.
